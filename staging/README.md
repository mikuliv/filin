# Staging-контур

`staging/` содержит локальные компоненты проверенного ранее пассивного
транспортного контура. Они не означают доступность реального staging
окружения и не разрешают внешнее подключение.

Текущие ограничения: backend integration, shadow mode, production,
уведомления и автоматическое применение решений запрещены. Назначение
каталога и поток доставки описаны в
[архитектуре delivery runtime](../docs/architecture/delivery-runtime.md).
