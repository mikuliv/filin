# Филин v0.3.13 — prospective blind environmental holdout

Этап проверяет неизменённый candidate `v0311:19176acb401be2d4` на новых условиях среды. До сбора были заморожены protocol, десять run IDs и seeds, сценарии, 25 benign variants, episode schedule, environmental profiles, policies и план regression bundle.

## Кампания и данные

В изолированных Docker-сетях завершены 10/10 прогонов пяти групп. Собрано 760 уникальных PCAP: 60 warm-up и 700 scored windows. В выборке 200 эпизодов, поровну benign и attack; каждый attack-класс имеет 20 эпизодов, каждый benign variant — четыре, длины 2/3/4/5 представлены по 50 раз.

## Blind-контур

Capture manifest и input lock созданы до inference. Таблица содержит 51 frozen feature и не содержит labels. Label vault запечатан до единственной immutable prediction. Fit, calibration fit, conformal fit, threshold selection, feature selection и candidate replacement не выполнялись.

## Результат

Window macro F1 равен `0.967762`, balanced accuracy `0.965238`, benign recall `0.991429`, FPR `0.008571`, attack macro recall `0.960000`, attack macro F1 `0.963593`. Attack episode recall и episode alert precision равны `1.0`, benign episode false-alert rate — `0.0`, detection by first/second/third window — `0.97/1.0/1.0`. Семь causal-order профилей дали точное совпадение.

Все frozen gates и строгая проверка regression bundle пройдены. v0.3.13 разрешает проектирование v0.3.14 shadow-readiness. Сам shadow mode, backend integration и production readiness остаются запрещены. Holdout-строки нельзя использовать для tuning или повторного выбора candidate.
