# Филин v0.3.18 — проектирование независимой внешней проверки и слепого испытания

## Назначение

Этап определяет воспроизводимый процесс, по которому внешний специалист сможет
проверить frozen candidate на заранее не показанных данных. v0.3.18 создаёт
протокол, contracts, evaluator, package verifier и синтетическую репетицию, но
не выполняет реальное внешнее испытание.

## Замороженные границы

Candidate `v03154:65a3dd912d845bc1`, candidate registry, feature/event/timing
contracts, state policy и backend остаются неизменными. Fit, calibration,
threshold selection и feature selection запрещены.

В этапе разрешён только режим `synthetic_protocol_rehearsal`. Реальные внешние
данные, реальные labels, внешняя организация, сетевые подключения, shadow mode,
backend integration, production и automatic enforcement не используются.

## Два будущих режима данных

`frozen_external_evaluation` предназначен только для независимой проверки
текущего кандидата. `authorized_development` создаёт новую candidate lineage и
лишает dataset статуса blind holdout. Одновременное назначение обоих режимов
запрещено.

## Последовательность blind workflow

Dataset и labels получают commitments до передачи blind inputs оператору.
Candidate и evaluator замораживаются до inference. Labels раскрываются только
после prediction commitment. Затем frozen evaluator рассчитывает заранее
определённые metrics, а reviewer проверяет roles, hashes и chronology.

SHA-256 в этом процессе является hash commitment, а не цифровой подписью.
Отрицательный результат или invalidation сохраняется и не заменяется повторной
попыткой без отдельной revision.

## Readiness

Положительный результат v0.3.18 разрешит только v0.3.19 — независимый review
пакета и согласование trial plan. Он не разрешит фактическое внешнее испытание
или эксплуатацию.
