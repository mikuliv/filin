# Ограничения архитектуры

- Current pipeline проверен только в локальном лабораторном и synthetic scope.
- Реальная организационная accuracy неизвестна.
- Фактический external blind trial не выполнялся.
- Backend и current sensor/runtime path не интегрированы.
- Real shadow mode, production capture и automatic enforcement запрещены.
- Reference receiver не заменяет operational backend.
- Evidence validators подтверждают заявленные artifacts, но не устраняют риски
  вне проверенного scope.
- Планируемая v0.4.x architecture не является current capability.
