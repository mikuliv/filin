# Выделение проекта в отдельный репозиторий

Платформа «Филин» первоначально разрабатывалась в каталоге `filin/`
репозитория исторического проекта Anomalyzer.

11 июля 2026 года история каталога была выделена в самостоятельный
репозиторий с помощью фильтрации Git. В новую историю включены только
коммиты и изменения, относящиеся к каталогу `filin/`.

Исходные идентификаторы коммитов технически изменились из-за изменения
деревьев Git. Карта соответствия исходных и новых коммитов сохранена
владельцем проекта вне репозитория.

Код, модели, datasets и runtime-компоненты исторического Anomalyzer
в новый репозиторий не переносились.

Результаты технического аудита происхождения кода находятся в разделах:

- [Аудит лицензирования](licensing-audit.md);
- [Аудит происхождения кода](code-origin-audit.md);
- [Лицензии зависимостей](dependency-licenses.md);
- [План разделения репозиториев](repository-separation-plan.md).

Документ описывает техническую миграцию и не устанавливает окончательный
юридический статус или правообладателя.
