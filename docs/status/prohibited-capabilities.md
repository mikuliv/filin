# Запрещённые и неподтверждённые возможности

Current readiness не разрешает:

- фактическое external trial execution;
- trial с реальной организацией;
- real traffic capture;
- real shadow mode;
- backend или SIEM integration;
- production connection;
- automatic enforcement;
- network blocking;
- реальные notifications;
- утверждения о real-world accuracy;
- утверждения о внешней security/privacy validation.

Эти ограничения действуют независимо от положительного результата synthetic
rehearsal v0.3.18. Future capability нельзя описывать как «почти готовую».
