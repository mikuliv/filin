# Пакет независимой проверки

Этот каталог объясняет протокол blind evaluation без необходимости изучать всю
историю проекта. Пакет предназначен для проверки identities, commitments,
chronology, evaluator и требований к будущему trial.

Длительная локальная репетиция v0.3.17 выявила недостатки evidence tooling.
Корректирующий v0.3.17.1 классифицировал причины, не меняя delivery path, и
подтвердил исправленную instrumentation targeted trial. Ни один из этих этапов
не является внешней проверкой.

v0.3.18 использует только синтетическую protocol rehearsal. Реальные данные,
labels и организации не задействованы. Начать review следует с
[подтверждённого scope](confirmed_scope.md), [ограничений](known_limitations.md)
и [руководства reviewer](reviewer_guide.md).
