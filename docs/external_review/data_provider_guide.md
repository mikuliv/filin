# Руководство data provider

Provider подтверждает право передачи, происхождение, capture period, ownership,
privacy status и usage mode. Он формирует episode/time/node/environment/
organization/capture-origin groupings и dataset manifest. Разные имена файлов не
доказывают отсутствие overlap. PCAP с credentials, запрещённым payload,
непроверяемым origin или неподтверждённым legal basis отклоняется.
