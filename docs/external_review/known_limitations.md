# Известные ограничения

- На внешних данных не рассчитана ни одна научная метрика.
- Независимый reviewer ещё не проверял пакет.
- Реальный blind holdout отсутствует.
- Организационные false-positive/false-negative limits не определены.
- Универсальный минимальный sample size не назначен без контекста источника.
- Правовые checklist не заменяют заключение компетентного юриста.
- Package verifier проверяет целостность и процесс, но не доказывает
  репрезентативность dataset.
- Shadow mode, backend integration и production запрещены.

Для локальной репетиции обязательно: `scientific_evidence=false`.
