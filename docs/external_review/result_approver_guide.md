# Руководство result approver

Approver принимает review findings, chronology и aggregate result. Допустимые
исходы различают stopped, invalidated, scientific/operational failure и
completed pass/fail. Отрицательный результат сохраняется. Approver не может
совмещать эту роль с operator, custodian, evaluator или reviewer в реальном
испытании.
