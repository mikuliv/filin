# Сопоставление MITRE

Сопоставление MITRE в прототипе является первичной таблицей соответствий. Оно не заменяет расследование аналитика.

## Первичная таблица

| Класс | MITRE ATT&CK | Пояснение |
| --- | --- | --- |
| DDoS | T1498 Network Denial of Service | Высокая интенсивность пакетов или SYN может соответствовать отказу в обслуживании. |
| PortScan | T1046 Network Service Discovery | Много портов назначения похоже на разведку сетевых сервисов. |
| Brute Force | T1110 Brute Force | Повторяющиеся неуспешные попытки входа соответствуют подбору учетных данных. |
| Web Attack | T1190 Exploit Public-Facing Application | Подозрительные web-признаки могут указывать на эксплуатацию публичного приложения. |
| Botnet | T1071 Application Layer Protocol | Осторожный кандидат: beacon-поведение нужно подтверждать DNS, HTTP или TLS-логами. |

## Ограничение

Класс модели не доказывает технику ATT&CK. Сопоставление должно использоваться как подсказка для расследования и формулирования гипотезы.
