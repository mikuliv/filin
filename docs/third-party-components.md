# Сторонние компоненты

| Компонент | Назначение | Источник и способ использования | Лицензия |
| --- | --- | --- | --- |
| Docker | Изолированный лабораторный runtime и named volumes | Внешний runtime, код не включается в репозиторий | Уточняется по дистрибутиву Docker |
| Zeek | Offline обработка фактического PCAP | Внешний container image/runtime, код Zeek не включается | Уточняется по официальному дистрибутиву Zeek |
| scikit-learn | Pipeline baseline evaluation | Python dependency проекта | Уточняется по файлам зависимостей и upstream |

Этот список описывает техническое использование, а не даёт юридический вывод. Условия распространения новой части проекта требуют отдельного уточнения. Наличие лицензии исторического компонента не означает автоматическое применение той же лицензии ко всем новым материалам без анализа структуры производного произведения.
