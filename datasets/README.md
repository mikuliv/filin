# Датасеты Филин

В этой папке хранится только описание структуры датасетов и служебные пустые файлы.

Большие датасеты, PCAP, сырые логи, JSONL-файлы, CSV-выгрузки, модели и артефакты обучения не хранятся в Git. Они сохраняются локально, размещаются в защищенном хранилище или публикуются отдельно с описанием версии, источника и методики сбора.

Рекомендуемая локальная структура:

```text
filin/datasets/
  raw/          # сырые логи и PCAP
  processed/    # нормализованные JSONL/CSV
  reports/      # отчеты по прогонам
```

Перед публикацией датасета нужно проверить отсутствие секретов, внешних адресов, персональных данных и лишних технических артефактов.
