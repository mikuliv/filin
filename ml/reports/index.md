# Индекс исследовательских отчётов

Tracked отчёты содержат агрегированное evidence без PCAP, SQLite, raw timing
traces, локальных абсолютных путей и закрытых ключей.

- `v0_3_17/` — неизменённый отрицательный итог четырёхчасовой controlled local
  rehearsal и post-campaign correction.
- `v0_3_17_1/` — corrective audit: проверка SSD migration, root cause десяти
  anchors и 69 806 timestamp violations, семантика transport duplicates,
  prospective latency/performance, corruption 20/20, finalizer, readiness и
  claim-evidence ledger.

Raw evidence хранится только в project-scoped runtime и не является частью
Git bundle.
