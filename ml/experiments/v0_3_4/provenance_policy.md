# Provenance policy

Every future v0.3.4 dataset and artifact must record campaign role, run IDs,
scenario parameters, PCAP and Zeek hashes, ordered feature list, environment
versions, model parameters, and artifact SHA-256. Marker fields, raw endpoints,
labels, seeds, and execution identifiers are metadata and forbidden model
features. No v0.3.5 record may be used during v0.3.4 development.
