# Машиночитаемые контракты внешней проверки

`external_review/contracts/` содержит 13 JSON Schema contracts frozen
workflow v0.3.18. Схемы являются частью подготовленного пакета и не
подтверждают проведение реального внешнего trial.

Человекочитаемая навигация, роли и ограничения находятся в
[пакете независимой проверки](../docs/external_review/README.md), а индекс
контрактов — в [документации контрактов](../docs/contracts/index.md).
