# Control API

`control-api` - учебный heartbeat/control endpoint для проверки периодичности обращений внутри лабораторного стенда Филин v0.1.

Сервис не имитирует реальный канал управления и не предназначен для вредоносных сценариев. Он принимает только безопасные beacon-запросы от лабораторного traffic-client внутри Docker-сети.
